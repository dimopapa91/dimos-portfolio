# Dimos Papageorgiou — Portfolio

Personal portfolio site for Dimos Papageorgiou, Data Engineer & Music Producer.

Built with vanilla Three.js, HTML & CSS. No build step required — just open `index.html` in a browser.

## Features

- Animated 3D terrain background (Three.js + Simplex noise shader)
- Dark amber colour palette
- Blend-mode cursor effect
- Scrollable project list with hover interactions
- Film grain overlay

## Projects featured

- **Waveline** — Music analytics web app (Flask · Python · Spotify API)
- **Blakenor** — Ambient & electronic music project
- **Music Insight Pipeline** — Spotify data pipeline (Python · ETL)

## Running locally

Just open `index.html` directly in your browser, or serve with:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`

## Deployment

Deploy to GitHub Pages, Netlify, or Vercel — all work with a static HTML file.
